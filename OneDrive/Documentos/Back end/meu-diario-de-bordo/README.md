### 🐳 INFRAESTRUTURA (Docker)
| Ação | Comando | O que faz? |
| :--- | :--- | :--- |
| **Ligar o motor** | `docker-compose up -d` | Sobe o Redis e o MongoDB de uma vez. |
| **Desligar tudo** | `docker-compose down` | Para os serviços e limpa a casa. |

### 🛠️ DIAGNÓSTICO (Resilience Toolkit)
| Ação | Comando | O que faz? |
| :--- | :--- | :--- |
| **Passar o Scanner** | `node doctor.js` | Verifica se os bancos estão Online ✅ ou Offline ❌. |

### 🐙 GIT & GITHUB (Sincronização)
| Ação | Comando | O que faz? |
| :--- | :--- | :--- |
| **Verificar Status** | `git status` | Mostra o que você mudou e se está "travado". |
| **Preparar e Salvar** | `git add .` && `git commit -m "..."` | Empacota suas mudanças com uma mensagem. |
| **Enviar para Nuvem** | `git push origin main` | Faz o seu "tapete verde" brilhar no GitHub. |
| **Mudar de Rota** | `git branch -M main` | Garante que você está na trilha principal (main). |

### ⌨️ ATALHOS SALVA-VIDAS

* **Ctrl + L**: Limpa a sujeira do terminal para focar no novo.
