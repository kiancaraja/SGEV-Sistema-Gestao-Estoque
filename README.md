Polos: Igarapava, Barrinha e Pitangueiras

📝 Resumo do Projeto
Desenvolvimento de um sistema digital para controle de medicamentos em unidades de acolhimento (ILPIs). O software visa registrar a validade dos produtos e emitir alertas automáticos de vencimento e estoque baixo, reduzindo desperdícios.
