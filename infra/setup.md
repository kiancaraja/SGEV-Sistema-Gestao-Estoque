🏗️ Infraestrutura e Configuração do Ambiente - SGEV
Este documento descreve as ferramentas necessárias para rodar o projeto localmente.

📋 Pré-requisitos
Java JDK: Versão 17 ou superior (Padrão Univesp).

Node.js: Versão 18 ou superior.

Banco de Dados: MySQL ou PostgreSQL (a definir).

IDE Recomendada: VS Code com extensões para Java.

⚙️ Como configurar
Instalar as dependências do Node: npm install

Configurar as variáveis de ambiente no arquivo .env (será criado em breve).
